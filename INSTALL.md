# INSTALL

## Introduction to C Programming

This guide explains how to set up, build, and run the project on your local machine.

---

## 1. Prerequisites

### 1.1 Required Software

- **C Compiler**
  - Recommended:
    - **GCC** (Linux / macOS / Windows via MinGW or WSL)

- **Text Editor or IDE**
  - Visual Studio Code `(Recommended)`
  - Code::Blocks
  - Dev-C++
  - CLion
  - Any plain text editor

### 1.2 Supported Operating Systems

- Windows
- Linux `(Recommended)`
- macOS

---

## 2. Installation

### 2.1 Clone the Repository

Clone the repository to your local machine:

```bash
git clone https://github.com/Computer-Programming-aka-Uniwa/Intro.git
```

### 2.2 Alternative (Without Git)

- Open the repository URL in your browser
- Click Code → Download ZIP
- Extract the ZIP file to a local directory

---

## 3. Compiling & Running the Programs

### 3.1 Navigate to Source Directory

```bash
cd Intro/src
```

### 3.2 Compile the Correct Program

Using GCC:

```bash
gcc C1aCorrect.c -o C1aCorrect
```

### 3.3 Run the Program

Linux / macOS:

```bash
./C1aCorrect
```

Windows:

```bash
C1aCorrect.exe
```

---

## 4. Open the Documentation

1. Navigate to the `docs/` directory
2. Open the report corresponding to your preferred language:
   - English: `Introduction.pdf`
   - Greek: `Εισαγωγή.pdf`
